# event-management

イベント管理システムのリポジトリです。

## 環境

- PHP8.2
- Laravel 10
- React 18
- Next13
- Typescript

## 環境構築

下記の流れに従って、環境構築を行なってください。
尚、SPA なのでフロントエンドも環境構築も行う必要があります。

#### clone

```
git clone git@github.com:yusuke-takatsu/event-management.git
```

#### install

```
make install
```

## push 前にやること

下記コマンド実行し、コードがフォーマットされていることを確認

```
make format
```
